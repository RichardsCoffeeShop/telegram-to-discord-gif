# Telegram Gifs For Discord!

## Run With Docker

`1. create .env file and set configs`

`2. docker-compose up`

## run Without Docker

`1. install node x16`

`2. install git`

`3. git clone https://github.com/sajjadmrx/telegram-to-discord-gif.git`

`4. npm install`

`5. create .env file and set configs`
[example](.env.ex)

`6. npm run start`
